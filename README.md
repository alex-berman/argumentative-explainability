- Berman, A. (2024). [Argumentative Dialogue As Basis For Human-AI Collaboration](https://ceur-ws.org/Vol-3825/short3-2.pdf). In _Proceedings of the Communication in Human-AI Interaction Workshop (CHAI-2024)_.
